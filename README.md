# Intelligent Robotics - Group 8
## Members: Maryam Bahamadan and Jaya Sharma

Aim: Develop a robot guide dog for navigation and object detection

Files provided:
- Final.py: The controller code used for the robot dog
- car_controller.py: The controller code used for the moving gate obstacles
- spot_maze.wpt: used to test the robot's navigation abilities
- spot_complex_maze.wbt: The world file for the complex maze used to test the robot's navigational abilities
- spot_maze_with_moving_objects.wbt: used to test the robot's ability to navigate through moving objects
